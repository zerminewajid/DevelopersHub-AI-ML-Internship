# DevelopersHub Corporation - AI/ML Engineering Internship Tasks

##  Overview

This repository contains a comprehensive collection of **AI/ML Engineering tasks** completed during the internship program at **DevelopersHub Corporation**. The project covers diverse machine learning applications, from exploratory data analysis to advanced deep learning implementations.

**Program Duration:** Internship Tasks  
**Deadline:** 15th May, 2026

---

##  Project Structure

The repository is organized into **6 major tasks**, each demonstrating different aspects of machine learning and data science:

### **Task 1: Iris Exploratory Data Analysis (EDA)**
- **Objective:** Comprehensive exploratory analysis of the famous Iris dataset
- **Key Deliverables:**
  - Statistical summaries and descriptive analysis
  - Visualizations: Histograms, pair plots, correlation analysis, box plots
  - Insights into feature distributions and relationships
- **Technologies:** Python, Pandas, Matplotlib, Seaborn
- **Output:** Task1_iris_eda.ipynb + Visualizations

### **Task 2: Stock Price Prediction**
- **Objective:** Build a machine learning model to predict stock prices
- **Key Deliverables:**
  - Time-series feature engineering
  - Model training and evaluation
  - Actual vs. predicted price comparisons
  - Feature importance analysis
- **Technologies:** Python, Scikit-learn, Pandas, Matplotlib
- **Output:** Task2_stock_prediction.ipynb + Visualization plots

### **Task 3: Heart Disease Classification**
- **Objective:** Develop a classification model to predict heart disease risk
- **Key Deliverables:**
  - Binary classification using multiple algorithms
  - Feature correlation and importance analysis
  - ROC curves and confusion matrices
  - Model performance comparison
- **Technologies:** Python, Scikit-learn, Classification models
- **Output:** Task3_heart_disease.ipynb + ROC curves, Confusion matrices

### **Task 4: Health Chatbot**
- **Objective:** Build an interactive health-related chatbot application
- **Key Deliverables:**
  - Natural language processing implementation
  - Chatbot logic and response generation
  - User interaction interface
- **Technologies:** Python, NLP libraries
- **Output:** Task4_health_chatbot.ipynb

### **Task 5: Mental Health Chatbot with Deep Learning**
- **Objective:** Advanced chatbot using deep learning for mental health support
- **Key Deliverables:**
  - Neural network model development
  - Training on mental health conversations
  - Model loss curves and training metrics
  - Trained model saved for deployment
- **Technologies:** Python, TensorFlow/Keras, Deep Learning
- **Output:** Task5_mental_health_chatbot.ipynb + Trained model directory

### **Task 6: House Price Prediction**
- **Objective:** Predict house prices using regression techniques
- **Key Deliverables:**
  - Feature engineering and preprocessing
  - Regression model development
  - Price distribution analysis
  - Geographic price analysis
  - Actual vs. predicted comparisons
- **Technologies:** Python, Scikit-learn, Regression models
- **Output:** Task6_house_price.ipynb + Visualizations

##  Quick Start

### Prerequisites
```bash
Python 3.7+
pip (Python package manager)
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/DevelopersHub-AI-ML-Internship.git
cd DevelopersHub-AI-ML-Internship
```

2. Create a virtual environment (optional but recommended):
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

3. Install required dependencies:
```bash
pip install -r requirements.txt
```

### Running the Projects

Each task is contained in its own directory as a Jupyter Notebook:

```bash
# Navigate to a task
cd task1

# Open Jupyter Notebook
jupyter notebook task1_iris_eda.ipynb
```
