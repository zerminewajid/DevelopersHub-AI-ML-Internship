# Task 6: House Price Prediction

## Overview
This project analyzes housing data and builds a regression workflow to estimate house prices from available features.

## Features
- Performs data cleaning and exploratory analysis
- Studies price distribution and geographic patterns
- Trains a regression model for price prediction
- Includes residual-style evaluation visuals and feature importance analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Structure
- `task6_house_price.py` - main regression Script
- `task6_price_distribution.png` - target distribution plot
- `task6_geo_prices.png` - geographic price visualization
- `task6_correlations.png` - feature correlation heatmap
- `task6_feature_importance.png` - model feature importance
- `task6_actual_vs_predicted.png` - model prediction comparison

## Screenshots
The project folder includes visual outputs that can be used as screenshots in GitHub.

## How to Run
1. Open `task6_house_price.ipynb` in VS Code.
2. Install the required Python packages if needed.
3. Run the Script to reproduce the full analysis and model evaluation.
