# Task 1: Iris Exploratory Data Analysis

## Overview
This project explores the classic Iris dataset and summarizes how the flower measurements vary across species.

## Features
- Loads and inspects the Iris dataset
- Compares sepal and petal measurements across classes
- Includes histograms, pair plots, and correlation analysis
- Uses visual exploration to understand class separation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn


## Project Structure
- `task1_iris_eda.py` - main analysis script
- `task1_histograms.png` - distribution plots
- `task1_pairplot.png` - pairwise feature relationships
- `task1_correlation.png` - correlation heatmap
- `task1_boxplots.png` - class comparison box plots
